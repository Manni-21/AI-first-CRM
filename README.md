# AI-First CRM – HCP Interaction Module

## 📌 Project Overview
AI-powered CRM system for logging Healthcare Professional interactions.

## 🚀 Features
- Structured form
- AI chat interface
- LangGraph AI extraction

## 🧠 AI Functionality
Input:
"Met Dr Sharma at Apollo hospital"

Output:
{
  "hcp_name": "Dr Sharma",
  "hospital": "Apollo hospital",
  "notes": "Discussion details"
}

## 🛠 Tech Stack
- React
- FastAPI
- LangGraph
- Groq LLM

## ⚙️ How to Run

Backend:
cd backend  
uvicorn main:app --reload  

Frontend:
cd frontend  
npm start
